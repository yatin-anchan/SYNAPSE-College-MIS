## 📌 Overview

**SYNAPSE** is a modern College Management Information System (MIS) designed to centralize and streamline academic and administrative operations.

---

## 🎯 Key Features

- Role-based access (Student, Faculty, HOD, COE)
- Attendance & analytics system
- Exam management & result generation
- Real-time Firebase sync
- Offline support

---

## 🏗️ Tech Stack

- Kotlin (Android)
- Firebase (Auth + Firestore)
- Material UI

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/synapse-mis.git
```

Open in Android Studio → Add Firebase config → Run
